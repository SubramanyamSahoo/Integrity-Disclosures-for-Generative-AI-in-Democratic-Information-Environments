# Failure-Mode Governance for Generative AI

Interactive public-facing research website for **Integrity Disclosures for Generative AI in Democratic Information Environments**.

## Files

- `index.html` — website entry point; includes the single BibTeX citation block at the bottom
- `styles.css` — visual design and animations
- `script.js` — interactive cards, heatmap, compound-risk calculator, citation copy button
- `assets/` — original SVG visualizations
- `.nojekyll` — prevents GitHub Pages from processing the site as a Jekyll project

## GitHub Pages

- Source: Deploy from a branch
